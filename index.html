Estrutura básica

Um arquivo me HTML sempre se inicia com a chamada "estrutura básica", que é praticamente imutável

Geralmente os IDEs (Ambiente de Desenvolvimento Integrado) já possuem atalhos para iniciar o arquivo de HTML com a sua estrutura básica.

A estutura básica do HTML é simples de ser identificada e ela se divida em 3 blocos:

  1. Doctype
  2. Head
  3. Body

Doctype

Doctype é uma instrução de ajuda os browsers/navegadores a entenderem o arquivo em HTML, permitindo que consigam ler os códigos que constam no documento e integram o site.

É importante observar que o doctype não é uma tag de hmtl e devem estar presente no topo do arquivo

Head

O Head contém informações que não são visíveis aos usuários/clientes.

São dados implicitos de uso e controle do documento. Ficando limitado a abertura e fechamento da tag "head"

Body

É o corpo do documento, é tudo o que irá aparecer no site para o usuário/cliente.

É todo e qualquer texto e informação que se deseja apresentar, assim como toda e qualquer forma de mídia de saída, como imagens, sons, miniaplicativos embutidos, conteúdo multimídia, etc. Além disso, toda a apresentação de entrada de dados também se aplica neste seção do documento, como é o caso dos formulários. Na prática, o corpo do documento é delimitado pelo tags < body > e < / body >.

TAGs

As TAGs são elementos de HTML e são utilizadas para informar ao browser/navegador qual o tipo de estrutura que está sendo construída dentro do arquivo.

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.
